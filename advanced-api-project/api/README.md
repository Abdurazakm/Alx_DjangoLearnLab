## Book API Endpoints

- **List Books**  
  GET /api/books/  
  Returns a list of all books.

- **Retrieve Book**  
  GET /api/books/<id>/  
  Returns details of a single book.

- **Create Book**  
  POST /api/books/ (Auth required)  
  Creates a new book.

- **Update Book**  
  PUT /api/books/<id>/ (Auth required)  
  Updates an existing book.

- **Delete Book**  
  DELETE /api/books/<id>/ (Auth required)  
  Deletes a book.
